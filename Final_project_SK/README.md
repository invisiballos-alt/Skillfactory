# Электронная сервисная книжка для складской техники «Силант»

заказчик —  [Чебоксарский завод силовых агрегатов (ЧЗСА)](https://chzsa.ru/)
В 2021 году завод начал выпускать свои вилочные погрузчики. Бренд назвали «Силант»

Ключевая сущность — «машина
Содержит информацию о характеристиках и комплектации проданных машин, а также информацию о месте эксплуатации.
